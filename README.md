# 📊 Sales Data Analysis using Python (EDA Project)

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on a retail sales dataset using Python.  
The main objective is to analyze sales performance, profitability, customer trends, and regional business performance using data analysis and visualization techniques.

The project helps identify:
- High-performing product categories
- Top-selling products
- Regional sales trends
- Profitability patterns
- Impact of discounts on profit

---

## 🛠️ Tools & Technologies Used
- Python 🐍
- Pandas 📊
- NumPy 🔢
- Matplotlib 📈

---

## 📂 Dataset Information
The dataset contains retail sales transaction data with the following information:

- Order ID & Order Dates
- Customer Details
- Product Categories & Sub-Categories
- Sales, Profit, Quantity, Discount
- Region, State, and City Information

Dataset Size:
- 9,994 Rows
- 21 Columns

## 🔍 Project Workflow

### 1️⃣ Data Loading
- Imported dataset using Pandas
- Loaded CSV file into DataFrame

### 2️⃣ Data Exploration
- Checked dataset structure
- Analyzed column data types
- Verified dataset dimensions

### 3️⃣ Data Cleaning
- Checked missing values
- Converted date columns into datetime format

### 4️⃣ Exploratory Data Analysis (EDA)
Performed:
- Total Sales Analysis
- Total Profit Analysis
- Average Sales Analysis
- Category-wise Sales & Profit Analysis
- Product Performance Analysis
- State-wise Sales Analysis
- Monthly Sales Trend Analysis

### 5️⃣ Data Visualization
Created visualizations using Matplotlib:
- Bar Charts
- Line Charts
- Scatter Plots
